## Quick Checklist for Every Page

- [ ] **Visual first**: Diagram/chart visible BEFORE reading
- [ ] **Text alongside**: Not after, ALONGSIDE the visual
- [ ] **Code immediately**: Example shown right after concept
- [ ] **Run + Output**: Show what happens when code executes
- [ ] **Chunked**: No paragraph > 80 words
- [ ] **Spaced**: Lots of whitespace
- [ ] **Font**: Sans-serif, 16pt+, NOT justified
- [ ] **Color**: High contrast, color-not-only meaning
- [ ] **Challenge**: At least one challenge included

## Font & Spacing Rules

### Typography

```
Heading:     32pt, bold, Arial
Subheading:  24pt, bold, Arial
Body:        16pt, regular, Arial
Code:        14pt, monospace, Courier
```

### Line Height & Spacing

```
Body text:      1.5x line height
Code blocks:    1.2x line height
Paragraphs:     1rem (16px) spacing
Sections:       2-3rem (32-48px) spacing
```

### DO & DON'T

```
âœ… DO:
- Sans-serif fonts
- Bold for emphasis
- Letter-spacing: +0.12em (optional toggle)
- Ample white space
- Color + texture (not color alone)

âŒ DON'T:
- Italic text in large blocks
- Justified paragraphs
- All-caps (except acronyms)
- Dense paragraphs (> 5 lines)
- Colored text without background
- Flashing or auto-playing content
```
