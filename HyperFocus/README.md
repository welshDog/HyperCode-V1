# HyperFocus

A hyperfocus productivity tool to help you stay focused and productive.

## Installation

```bash
# Install from source
git clone https://github.com/yourusername/hyperfocus.git
cd hyperfocus
pip install -e .[dev]  # For development
# or
pip install .  # For production
```

## Usage

```bash
hyperfocus --help
```

## Development

1. Clone the repository
2. Install with development dependencies:
   ```bash
   pip install -e .[dev]
   ```
3. Run tests:
   ```bash
   pytest
   ```
4. Format code:
   ```bash
   black .
   isort .
   ```
